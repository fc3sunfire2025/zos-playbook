# Glossary

A lightweight glossary to keep terminology consistent.

- **LPAR**: Logical Partition.
- **TSO**: Time Sharing Option.
- **ISPF**: Interactive System Productivity Facility.
- **SDSF**: System Display and Search Facility.
- **JCL**: Job Control Language.
- **DD**: Data Definition statement in JCL.

Add terms as you write docs: one-line definition + optional notes.
