# Disclaimer & safety notes

This repository is provided for educational and reference purposes.

## Review before execution

- **Review all code before execution.** Do not run code or commands you do not fully understand.
- **Test in a non-production environment first.** Prefer a sandbox LPAR or equivalent controlled environment.
- **Validate inputs and assumptions.** Many examples use placeholders that must be adjusted for your site.

## Security & compliance

- **Follow your site security processes.** Ensure changes comply with your organization’s security policies, change control, and audit requirements.
- **Least privilege.** Use the minimum permissions required for a task.
- **Sensitive data.** Do not commit credentials, personal data, or proprietary configuration.

Content is provided “as-is” without warranties of any kind. You are responsible for validating that any procedure, configuration, or code is safe and appropriate for your environment.
